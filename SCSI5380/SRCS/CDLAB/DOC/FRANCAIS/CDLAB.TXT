[MAJ] CDLab 0.91 disponible.

Mercredi, 2 Mai 2007,


CDLab 0.91 est disponible.
http://doyeuxyvan.free.fr/cdlab/v0.91/CDLAB091.ZIP
Un outil de gravure CD pour compatibles Atari.

--------------
Nouveautes
--------------
- Nouveaux formats pour l'extraction audio. ( AU/SND, AIFF, AIFF Cubase
Audio ).
- Creation d'un masque pour le nom des fichiers lors d'une extraction.
- Meilleure lisibilite en Monochrome.

-----------------
Fonctionnalites
-----------------
- Extraction de pistes CD audio.
- Effacement de CD-RW.
- Copie DAO (Disc-At-Once) pour tous les disques monosessions. ( Mais ca
ne marche pas sur mon graveur pourtant compatible MMC ! )
- TAO (Track-At-Once) mode multisession.




Vous avez besoin de l'interface SCSIDRV. ( deja incluse dans HDDriver )
Vous pouvez quand meme sinon le lancer avant CDLab:
http://doyeuxyvan.free.fr/scsidrv/SCSIDRV.PRG
